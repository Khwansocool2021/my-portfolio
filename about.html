เกี่ยวกับฉัน

สวัสดีครับ ผม ศุภโชค นิยม นักศึกษาที่มุ่งเน้นการพัฒนาระบบเครือข่าย (Networking) และการบริหารจัดการระบบ (Systems Administration) ด้วยความหลงใหลในการเชื่อมต่อโลกดิจิทัลเข้ากับอุปกรณ์ทางกายภาพ (IoT) ทำให้ผมมีประสบการณ์ตรงในการพัฒนาโปรเจกต์ ตั้งแต่ IT Support และ Network การวางระบบ LoRa Mesh ไปจนถึงการจัดการ Cloud Infrastructure บน AWS และ True NAS การจัดเก็บช้อมูลภายในองค์กร

นี้คือบันทึกเส้นทางการเรียนรู้ของผม ที่ผสมผสานระหว่างทักษะด้าน Web Development, Network Lab และ IoT Implementation IT Support และการแก้ไขปัญหาอุปกรณ์อิเล็กทรอนิกส์ ผมได้เรียนรู้ที่จะสร้างสรรค์โซลูชันที่ไม่เพียงแค่ตอบสนองความต้องการของผู้ใช้งาน แต่ยังสามารถปรับตัวและพัฒนาได้ตามเทคโนโลยีที่เปลี่ยนแปลงอย่างรวดเร็ว ผมเชื่อมั่นในการสร้างสรรค์เทคโนโลยีที่ไม่ได้เป็นเพียงแค่โค้ด แต่เป็นโซลูชันที่นำไปใช้งานได้จริง เพื่อยกระดับประสิทธิภาพและความก้าวหน้าทางเทคโนโลยีครับ
